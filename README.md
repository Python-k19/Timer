## 📖 О проекте

**Beautiful Timer** — это мощный и стильный инструмент для управления временем с расширенными возможностями автоматизации. Приложение сочетает в себе простоту использования и профессиональный функционал.

### ✨ Особенности

- 🎨 **Современный UI** — тёмная тема с плавными анимациями
-  **Два режима** — обратный отсчёт и точное время
- 🤖 **Автодействия** — 7 типов автоматических действий
- 🎵 **Кастомные звуки** — свои мелодии для уведомлений
- 🔧 **Управление ПК** — выключение, перезагрузка, сон
- 📊 **Прогресс-бар** — визуальное отслеживание времени

## 🚀 Возможности

### 📋 Режимы работы

| Режим | Описание |
|-------|----------|
| **⏳ Обратный отсчёт** | Установите время в формате ЧЧ:ММ:СС |
| **🕐 Точное время** | Запуск действия в определённое время |
